## Відповіді на контрольні запитання

Наслідування: Це створення нового класу на основі існуючого для повторного використання коду (принцип "Is-a" / "Є кимось").

Virtual vs Abstract: virtual має код і його можна змінити, abstract не має коду і його треба реалізувати обов'язково.

Base: Це слово для звернення до батьківського класу (щоб викликати його конструктор або метод).

Поліморфізм (Runtime): Це коли програма під час роботи сама вирішує, чий метод викликати (базовий чи перевизначений), дивлячись на реальний тип об'єкта.

Композиція vs Наслідування: Наслідування — це жорсткий зв'язок "є кимось" (Собака є Твариною). Композиція — це гнучкий зв'язок "має щось" (Машина має Двигун).

### Висновок: На цій лабораторній я навчився створювати ієрархії класів (Parent -> Child), використовувати base для конструкторів та реалізовувати поліморфізм через virtual/override, що дозволяє працювати з різними об'єктами через один список.
