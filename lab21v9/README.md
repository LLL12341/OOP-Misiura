# Лабораторна робота №21: Застосування принципу OCP через патерни Factory та Strategy

## 1. Загальна інформація
- **Тема:** OCP: гнучкі алгоритми розрахунку.
- **Мета:** Реалізація системи з використанням принципу Open/Closed (відкритості/закритості), що дозволяє додавати нові типи розрахунків без модифікації існуючого ядра програми.

---

## 2. Опис архітектури

Проєкт реалізовано з використанням двох ключових патернів проектування:

1.  **Strategy (Стратегія):** Використовується для інкапсуляції алгоритмів розрахунку вартості доставки. Кожна стратегія реалізує інтерфейс `IShippingStrategy`.
2.  **Factory Method (Фабричний метод):** Клас `ShippingStrategyFactory` відповідає за створення екземплярів конкретних стратегій на основі вхідних даних від користувача.

### Структура проекту:
- `IShippingStrategy` — спільний інтерфейс для всіх алгоритмів.
- `StandardShippingStrategy`, `ExpressShippingStrategy`, `InternationalShippingStrategy` — конкретні реалізації логіки.
- `DeliveryService` — клас-контекст, який працює з абстракцією, не знаючи про деталі реалізації.
- `ShippingStrategyFactory` — централізована точка створення об'єктів.

---

## 3. Дотримання принципів SOLID

### SRP (Single Responsibility Principle)
Кожен клас стратегії відповідає виключно за одну математичну формулу розрахунку. `DeliveryService` відповідає лише за виклик розрахунку, а не за логіку вибору тарифу.

### OCP (Open/Closed Principle)
Це головний фокус роботи. Система продемонструвала свою гнучкість при додаванні **NightShippingStrategy**:
- **Відкритість:** Ми змогли додати нову логіку (нічну доставку), створивши новий клас.
- **Закритість:** Нам не довелося змінювати код у класі `DeliveryService` або в існуючих стратегіях.

## 4. Приклади реалізації розрахунків

| Тип доставки    | Формула розрахунку                                   |
|-----------------|------------------------------------------------------|
| **Standard** | `distance * 1.5 + weight * 0.5`                      |
| **Express** | `(distance * 2.5 + weight * 1.0) + 50`               |
| **International**| `(distance * 5.0 + weight * 2.0) + 15% tax`          |
| **Night** | `(Standard Rate) + 100` (Додано через розширення)    |
